Introdução

Análise de Dados dos Voos entre Aeroportos Brasileiros para o Ano de 2023

Rotas Aéreas por Aeroporto

Esse gráfico iterativo tem como objetivo representar um gráfico entre os aeroportos públicos brasileiros. A proporção dos nós é calculada conforme a quantidade de rotas entre o aeroporto e suas conexões, portanto, quanto maior o tamanho do nó, maior é a quantidade de destinos.
